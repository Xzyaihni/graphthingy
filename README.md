# graphthingy
this sucks BUT it does show the units (i added them)

```
git clone https://github.com/Xzyaihni/graphthingy
cd graphthingy
cargo b -r
```


then run with
```
./target/release/graphthingy coolgraph1.txt coolgraph2.txt
```

each file should be in this format
```
step 1234
1.2345
5.4321
step 5
2.3456
```
